# 部落冲突阵型展示网站 - 设计文档

## [S1] 项目概述

个人部落冲突阵型收藏展示网站，用于展示、分类、分享自己设计的阵型。采用游戏风格还原的UI设计，让访问者仿佛在游戏内浏览阵型。

## [S2] 技术栈

- **框架**: Astro 5.x（静态输出模式）
- **CMS**: Sanity（Content Lake + Studio）
- **样式**: Tailwind CSS 4 + 自定义游戏风格组件
- **部署**: Vercel / Netlify
- **语言**: TypeScript

## [S3] 页面结构

```
/                    → 首页（精选阵型 + 快速筛选）
/bases               → 阵型列表页（完整筛选/搜索/排序）
/bases/[slug]        → 阵型详情页（大图 + 说明 + 分享）
/about               → 关于页
```

## [S4] 数据模型

### Sanity Schema - BaseLayout

```typescript
{
  name: string           // 阵型名称
  slug: Slug             // URL友好标识
  thLevel: number        // 大本营等级 (9-16)
  usage: enum            // 用途：war | farm | trophy | hybrid
  description: text      // 阵型说明/解析
  image: Image           // 阵型截图（Sanity asset）
  tags: string[]         // 自定义标签（如：原创、联赛、对称）
  isFeatured: boolean    // 是否精选
  createdAt: datetime    // 创建时间
  copyLink?: string      // 游戏内复制链接（可选）
}
```

### 用途枚举

| 值 | 中文 | 说明 |
|----|------|------|
| war | 部落战 | 专为部落战设计的防御阵型 |
| farm | 护资源 | 重点保护资源建筑 |
| trophy | 护杯 | 重点保护奖杯，防止被三星 |
| hybrid | 综合 | 兼顾资源和奖杯 |

### UI组件设计

**阵型卡片：**
- 木纹边框包裹
- 底部信息栏：阵型名称 + TH等级徽章 + 用途标签
- 悬停效果：轻微放大 + 金色光晕

**筛选栏：**
- 仿游戏内标签页设计
- TH等级：圆形徽章，带等级数字
- 用途：图标+文字标签
- 选中态：金色高亮 + 底部指示条

**详情页面板：**
- 仿游戏UI的石质面板
- 左侧大图，右侧信息面板
- 分享按钮：金属质感按钮

**导航栏：**
- 游戏顶部栏风格
- 网站logo + 导航链接
- 深色/浅色模式切换按钮

### 动画效果

- 页面加载：元素依次淡入（staggered reveal）
- 卡片悬停：轻微放大 + 光晕
- 背景：草地纹理 + 缓慢飘动的云朵
- 按钮：按下下沉效果 + 金属光泽

## [S6] 功能需求

### 核心功能

1. **阵型展示**：以卡片网格形式展示所有阵型
2. **分类筛选**：按TH等级、用途筛选
3. **搜索**：按阵型名称搜索
4. **排序**：按创建时间、精选优先排序
5. **详情查看**：点击卡片进入详情页，查看大图和说明
6. **分享**：生成分享链接，可复制到剪贴板
7. **深色/浅色模式**：支持主题切换，记住用户偏好

### 非功能需求

1. **性能**：静态页面，首屏加载 < 2秒
2. **SEO**：每个页面有独立的meta标签
3. **响应式**：适配桌面、平板、手机
4. **可访问性**：基本的ARIA标签和键盘导航

## [S7] 内容管理

### Sanity Studio

- 独立的Studio部署路径（/studio 或独立域名）
- 可视化编辑阵型信息
- 图片上传和管理
- 批量操作支持

### 数据流

```
Sanity Studio（后台编辑）
    ↓ webhook 触发构建
Astro 构建时拉取最新数据
    ↓ 静态生成
HTML/CSS/JS 输出 → 部署到 Vercel/Netlify
```

## [S8] 部署方案

- **前端**：Vercel 或 Netlify（静态托管）
- **CMS**：Sanity Cloud（免费额度足够个人使用）
- **域名**：自定义域名（可选）
- **构建**：Sanity webhook 触发 Astro 重新构建
